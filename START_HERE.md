## ⚡ QUICK START (5 MINUTES)

### Option 1: Deploy FREE on Replit (Easiest)

**Step 1: Push to GitHub**
```bash
git init
git add .
git commit -m "X Twitter Clone"
git remote add origin https://github.com/YOUR_USERNAME/x-twitter-login-clone
git push -u origin main
```

**Step 2: Go to Replit**
1. Visit https://replit.com (sign up free, no credit card)
2. Click "Create Repl"
3. Choose "Import from GitHub"
4. Paste: `https://github.com/YOUR_USERNAME/x-twitter-login-clone`
5. Click "Import"

**Step 3: Add Secrets**
1. Click "Secrets" 🔒 (left sidebar)
2. Add these:
```
SECRET_KEY=your-secret-key-here
FLASK_ENV=production
MAIL_SERVER=smtp.gmail.com
MAIL_USERNAME=your-email@gmail.com
MAIL_PASSWORD=your-app-password
```

**Step 4: Run**
- Click "Run" button
- Wait 30 seconds
- Your app is LIVE! 🎉

**Cost: $0**
**Time: 5 minutes**
